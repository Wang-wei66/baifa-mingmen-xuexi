# 《大乘百法明门论》互动学习 App

一个**单文件、纯离线**的百法学习工具，基于世亲菩萨造、玄奘法师译的《大乘百法明门论》，将「五位百法」的教理以可交互的方式呈现，帮助日常修学与自省。

## 特点

- **零依赖、零联网**：单个 `baifa-learning.html` 文件，双击即可在任意现代浏览器打开，所有数据与逻辑均内嵌其中。
- **五大板块**（底部导航切换）：
  1. **百法** —— 五位百法层级详学：心法（八识）、心所法（五十一，分遍行／别境／善／根本烦恼／随烦恼／不定六位）、色法（十一）、心不相应行法（二十四）、无为法（六）。每个法含定义、关联及包含、生活场景、转换与应对。
  2. **贯通** —— 法理脉络逻辑图（五位收摄、八识种子现行、心王心所相应、善烦恼对治、转识成智），点击任意逻辑图可**展开其聚合节点所含的具体诸法**，并附详细说明文字。
  3. **重点** —— 核心法理折叠卡片，含通俗解释、法理说明与生活比喻。
  4. **场景** —— 生活情境下的百法应用自检，分情境描述／百法分析／具体应用／修行启示四项。
  5. **考试** —— 473 道自测题，覆盖每一个法（重点法 ≥8 题，其余 ≥2 题）及各法之间的关联，可筛选、可跳转回百法详学。
- **暗色模式**：自动跟随系统配色，可手动切换。
- **响应式**：适配桌面与移动端。

## 使用方法

直接用浏览器打开 `baifa-learning.html` 即可，无需安装、无需服务器、无需联网。

```bash
# 如需本地起服务预览
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000/baifa-learning.html
```

## 内容说明

- 「五位百法」的框架依玄奘译《大乘百法明门论》，并参考唯识宗（瑜伽行派）相关注疏整理。
- 本应用为**学习辅助工具**，旨在帮助理解教理、用于日常心念观照与修行自检；文中「关联及包含」「生活场景」「考试题目」等为便于理解所作归纳，引用请以原典为准。
- 《大乘百法明门论》原文属公共领域；本应用的交互代码与整理内容可自由学习、转载，欢迎指出错漏。
